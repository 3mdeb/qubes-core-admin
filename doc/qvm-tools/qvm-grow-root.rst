=============
qvm-grow-root
=============

NAME
====
qvm-grow-root - increase root storage capacity of a specified VM

SYNOPSIS
========
| qvm-grow-root <vm-name> <size>

OPTIONS
=======
-h, --help
    Show this help message and exit
--allow-start
    Allow VM to be started to complete the operation
